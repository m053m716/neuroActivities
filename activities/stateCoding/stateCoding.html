Motor State Coding

Background

One of the challenges in understanding cortical control of movement is to understand the basis that it uses for computing and updating motor commands. This example is designed to illustrate how the movement characteristics of an effector change depending on the state variables controlling it. Here, the coordinate basis for the movement of the cursor on the screen can be toggled between cartesian and polar coordinates. You are a proxy for the nervous controller.

See also: Visuomotor Rotation Task (VMR)

Instructions

The cursor (yellow) begins in the starting position (gray circle). When the cursor turns blue, move it to the target (green circle). There will be 5 baseline trials followed by 10 perturbed trials followed by 5 washout trials.

  • Click the button with Cartesian or Polar to toggle the game mode.
  • Click Start to begin the game.
  • Navigate the cursor into the highlighted target by using the wasd keys.
  • Try changing the state basis and see how that influences ability to control the cursor.

Questions

  • How does the same VMR impact movements depending on the mode of the controller?
  • Are there different strategies that are better equipped at handling the perturbation?
  • Are there ways to remove strategies that prevent you from testing a hypothesis?
